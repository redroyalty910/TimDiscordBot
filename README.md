# TimBot

I made a discord bot to become more proficient in javascript... also made it because, well, I wanted to make my own bot. So I did! With a lot of help. Check it out! Invite it to your server, it's basically like I'm really there and stuff!

# FEATURES:

1. level-up system
2. level-up rank/leaderboard system
3. message-based XP system
4. XP cooldown system to prevent spam farming
5. economy system
6. passive daily income system
7. balance checking system
8. blackjack system
9. blackjack betting system
10. blackjack win/loss/stat tracking system
11. saved user data using JSON files

# FULL-LIST OF COMMANDS:

!hello - generates a hello message
!rank - informs the user of their current level
!leaderboard - generates the XP leaderboard
!balance - shows the users current moneys balance
!blackjack 100 - player can bet x amount of money and play blackjack
!hit - blackjack
!stand - blackjack
!blackjackstats - shows the users blackjack stats
